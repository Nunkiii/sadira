INAF/IASF-Bologna ☄ — Astro-web-software

Qk/Sadira is an experimental, scientific-oriented, computing application framework. At the time beeing, it is a mixed ECMAScript(JS)/C++ prototype running on Node.js servers and web browsers.

The goal of Sadira is to ease the setup of scientific data acquisition, processing and pipeline design tasks, from a practical scientific researcher point of view.

It will provide rich web browser application GUI based on an original HTML toolkit engine, server interfaces to databases, entry points for low-level, high performance data analysis/reduction algorithms written in Fortran/C/C++ or using the new possibilities offered by openCL.

Sadira is :

  • a cluster of Node.js servers synchronized with Redis,
  • an HTML5/CSS3/JavaScript/HTML toolkit based on JSON templates, skinned with Bootsrap CSS3 classes, fitting nicely in all screen sizes, and providing a fully browser-side construction of the HTML user interface,
  • a websocket/webRTC binary protocol for peer to peer data and processing exchange between servers and web-browsers,
  • serialization of user interface state in local websotorage or 'cloud-side', in mongo.DB documents,
  • a rich user management system, able to handle various login methods.
  • ...

Example experiments :

  • XD-1 FITS viewer
  • Webcam/DVD/Cardboard spectrograph
